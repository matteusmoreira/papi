# Pastorini API

WhatsApp API powered by Baileys - Gerencie múltiplas instâncias do WhatsApp com uma API REST completa.

## Instalação Rápida

### Docker Swarm / Portainer (Recomendado)
Use o arquivo `docker-stack.yml` para deploy em produção.

### Docker Compose
```bash
cp .env.example .env
docker-compose up -d
```

### Manual
```bash
npm install
npm start
```

## Documentação
Acesse `http://localhost:3000/docs.html` após iniciar o servidor.
